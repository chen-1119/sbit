# 测试清单（聚合版）

用于微信小程序发布前回归，覆盖深度主测 + 轻量模块聚合流程。

## 1. 环境准备

- 微信开发者工具：稳定版
- 基础库：默认 + 最近稳定版本
- 真机：iOS 1 台 + Android 1 台

## 2. 启动与首页

- [ ] 进入默认页 `pages/home/home`
- [ ] 首页统计模块数、题量、预计时长显示正常
- [ ] 分类筛选（全部/MBTI/职业/沟通/韧性/成长）切换正常
- [ ] 点击主测卡进入 `pages/index/index`
- [ ] 点击轻量模块进入 `pages/module-quiz/module-quiz?id=...`

## 3. 轻量模块流程（通用页）

### 3.1 答题页 `module-quiz`
- [ ] 题目、选项、进度条、预计时长显示正确
- [ ] 未选答案时点击下一题会提示
- [ ] 选项点击后高亮且可自动下一题
- [ ] 上一题可回看并保留答案

### 3.2 断点续测
- [ ] 答题中退出后重新进入，同模块可提示继续进度
- [ ] 选择“继续”后题号与答案一致
- [ ] 选择“重置”后从第 1 题开始

### 3.3 结果页 `module-result`
- [ ] 维度分布、主导画像、建议、城市气质标签显示正常
- [ ] “复制结果文案”可写入剪贴板
- [ ] “再测一次”可回到当前模块
- [ ] “返回测试中心”可回到首页

## 4. 深度主测流程（index/result）

- [ ] 43 题流程跑通
- [ ] 结果页显示 SBTI / MBTI / 城市匹配
- [ ] 城市职业偏好切换可即时重算
- [ ] 城市权重调节（1~4）可即时重算
- [ ] 海报生成、预览、保存到相册正常

## 5. 配置与编码检查

- [ ] `app.json` 可正常解析
- [ ] `app.wxss` 可正常编译
- [ ] `*.json/*.js/*.wxml/*.wxss` 编码均为 UTF-8 无 BOM
- [ ] 不存在 `app.json.json` 文件
- [ ] `app.json` 不包含无效 `permission["scope.writePhotosAlbum"]`

## 6. 兼容性与性能

- [ ] 首页与各结果页在小屏机型无溢出
- [ ] 页面切换与答题无明显卡顿
- [ ] 分享和复制功能在真机可用
