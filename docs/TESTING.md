# 测试清单（微信小程序）

该清单用于发布前回归。建议先在微信开发者工具执行，再做真机验证。

## 1. 测试环境

- 微信开发者工具：稳定版
- 基础库：使用项目默认 + 最近稳定版本各测一轮
- 终端：
  - iOS 真机 1 台
  - Android 真机 1 台（至少 1 台）

---

## 2. 核心流程回归

### 2.1 首页与启动
- [ ] 首页正常展示：标题、题量、流程、开始按钮
- [ ] 无历史进度时点击“开始测试”直接进入答题
- [ ] 有历史进度时弹窗可选“继续上次/重新开始”

### 2.2 答题过程
- [ ] 题目数量正确（43题）
- [ ] 进度条与百分比正常递增
- [ ] 预计剩余时间文本正常变化
- [ ] 单题未选时点击“下一题”出现提示
- [ ] 选项点击高亮和自动跳题正常
- [ ] 上一题可回看并保留已选答案

### 2.3 提交与结果页
- [ ] 最后一题后可进入结果页
- [ ] SBTI 类型、匹配度、描述显示正常
- [ ] MBTI 类型、清晰度、临界轴提示显示正常
- [ ] 城市 Top3、分轴匹配条、标签、建议显示正常

---

## 3. 断点续测验证

### 3.1 中断恢复
- [ ] 答题中退出小程序，再次进入可恢复进度
- [ ] 恢复后题号、已答题数、选项状态一致

### 3.2 清理逻辑
- [ ] 首页“清除进度”后不再出现恢复入口
- [ ] 提交结果后返回首页，旧进度不会复活

---

## 4. 城市偏好与权重验证

### 4.1 职业偏好
- [ ] 切换职业偏好（技术/金融/创意等）可即时重算 Top3
- [ ] 当前职业提示文案与筛选项一致

### 4.2 权重调节
- [ ] 四轴权重可在 1~4 区间增减
- [ ] 权重变化后城市排序与分数即时变化
- [ ] 预设切换（平衡/职业发展/机会社交/稳定可控）正常
- [ ] “恢复默认偏好”恢复到综合 + 平衡

### 4.3 偏好持久化
- [ ] 调整后退出再进，职业与权重能自动恢复

---

## 5. 分享与海报验证

### 5.1 文案复制
- [ ] 点击“复制文字版结果”成功写入剪贴板
- [ ] 文案包含：SBTI、MBTI、职业偏好、城市权重、城市Top3

### 5.2 海报生成
- [ ] 点击“生成分享图”可正常生成与预览
- [ ] 海报内容包含最新偏好与权重数据
- [ ] 生成后在页面可看到预览图

### 5.3 相册保存
- [ ] 首次保存触发授权逻辑
- [ ] 已授权后保存成功并有提示
- [ ] 拒绝授权时提示可引导到设置

---

## 6. 兼容性与稳定性

### 6.1 样式与布局
- [ ] iOS/Android 页面布局无错位
- [ ] 长文本、换行、按钮在小屏不溢出
- [ ] 海报文字在不同机型不截断

### 6.2 性能
- [ ] 首页首次打开不卡顿
- [ ] 答题切题无明显卡顿
- [ ] 结果页渲染完成时间可接受
